Healthcare operations team needed reliable Ubuntu server visibility

MedCare's internal applications depend on Ubuntu servers, but the support team needed a clearer way to track CPU, memory, disk usage, service status, log growth, and alert evidence during daily handovers.

I built a repeatable operations platform that provisions AWS infrastructure with Terraform, collects Linux health data with Bash checks, sends CloudWatch alarms through SNS email, and presents server status in a Dockerized Flask dashboard.

The project includes CI validation, Docker build evidence, Terraform validation, live EC2 dashboard screenshots, and operational artifacts that show how support engineers can detect issues earlier and document system health more clearly.
