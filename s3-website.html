Low-cost static website deployment using Amazon S3

The website needed a simple, reliable, and cost-conscious hosting solution that could serve static files publicly without managing a traditional server.

I configured Amazon S3 static website hosting, uploaded site assets, set index and error documents, adjusted bucket policies, and validated public access. I also documented the upload and access-control evidence so the deployment process can be repeated.

This project demonstrates practical cloud storage, website hosting, access policy troubleshooting, and AWS deployment skills.
